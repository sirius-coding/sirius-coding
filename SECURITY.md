# Security Policy

This repository is a public GitHub profile entry point. Treat every tracked file as public.

Do not commit credentials, tokens, keys, private hostnames, private IP addresses, sensitive paths, or login details.

Reusable engineering assets belong in `sirius-coding/sirius-evolution-station-template`. Concrete workstation and project details belong in adopter or project repositories, with private values kept in ignored local overlays or secret managers.

If sensitive data is found in a public repository, stop extending the affected file, remove or redact the value, rotate any exposed credential, and review whether history cleanup is required.
