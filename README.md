## Main Direction

`sirius-coding/sirius-coding` is now the public GitHub profile entry point.

The primary project is [Sirius Evolution Station Template](https://github.com/sirius-coding/sirius-evolution-station-template): a public upstream template for reusable Control Layer OS assets, including rules, skills, workflows, review standards, diagram capability, release evidence, and public/private guardrails.

The official adopter workspace is [Sirius Workstation](https://github.com/sirius-coding/sirius-workstation). It carries the full working station and concrete project experience, while reusable lessons should flow back into the template.

## Operating Model

The template is the upstream source of reusable experience. Any workspace created from it can contribute back durable improvements:

1. Use the template to start or organize a workspace.
2. Build real projects in an adopter workspace or independent repositories.
3. Promote repeated lessons into docs, skills, scripts, checks, diagrams, or workflows.
4. Contribute public-safe reusable improvements back to the template.

Business implementation, private environment values, and deployment details stay out of the template.
